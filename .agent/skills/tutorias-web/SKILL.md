## How to use it

1.  **Arquitectura:** Utiliza HTML5 semántico y CSS moderno (Flexbox/Grid). Mantén los archivos separados (`index.html`, `style.css`, `app.js`).
2.  **Estructura del Sitio:** Crea una página de Inicio atractiva, una sección de "Nuestros Servicios" detallando las materias, y un Formulario de Contacto funcional.
3.  **Seguridad y Mejores Prácticas:** Asegura que el formulario de contacto tenga validaciones estrictas de entrada en el frontend. Diseña el sitio con un enfoque "Mobile-First" para que sea completamente responsivo.
4.  **Flujo de trabajo:** Antes de codificar, presenta siempre un plan de implementación.
